# Papers

把你下载的 PDF 文献放在这里。

## 命名规范 (建议)

`[FirstAuthorLastName]_[Year]_[ShortTitle].pdf`

例如:
- `vanGils_2018_order_picking_systems_review.pdf`
- `Gademann_2005_order_batching_travel_time.pdf`
- `Cheng_2023_AMR_hospitals_stochastic.pdf`
- `Boysen_2019_warehousing_ecommerce_survey.pdf`

## Week 2 必读 (4 篇)
